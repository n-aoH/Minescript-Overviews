This is a collection of tutorials specifically for beginners wanting to get into minescript with little to no python experience.

It's self-paced, and introduces Python concepts alongside Minescript concepts. Each lesson assumes you have knowledge of the previous lesson.

For more information on python specifically, I recommend visiting [W3 schools](https://www.w3schools.com/python/default.asp). They provide tutorials and a guide on every python statement such as loops, more details on data, and more!

Take it as slow as you need to, messing around with programming is all the fun!
