# First Python File

## Creating The File

In order for your computer to recognize your script as something that python can execute, you must first create a file with a `.py` extension.

In your `minescript` folder (minecraft/mods/minescript/), right click in the folder to create a new text file.

<img width="521" height="421" alt="image" src="https://github.com/user-attachments/assets/0f02a1ec-d0b5-483e-871d-0ea8de7d4ea4" />

After doing this, you may not see the `.txt` extension. If you don't you must first enable `File Name Extensions` in windows.

<img width="597" height="20" alt="image" src="https://github.com/user-attachments/assets/0a092493-e48b-4238-b8f3-f67e085f7b41" />

<img width="1002" height="143" alt="image" src="https://github.com/user-attachments/assets/a34effa0-1e56-47b5-8323-3b716a48305a" />

You can now change the final extension from `.txt` to `.py`.

You will get a popup asking if you want to rename it, where you can click "yes"

<img width="598" height="174" alt="image" src="https://github.com/user-attachments/assets/95269d75-c886-4f1d-a888-263462b58fa0" />

Now, if you double-click the file, you can open it up in your preffered IDE.

The two most common IDEs used in the minescript community are VSCode and PyCharm. If you don't use an IDE, programming will be significantly more difficult.

Inside your python file, you can create the following code to print out a statement in python.

```python
print("Hello, world!")
```

Inside your game, you can now type `\` followed by your file name. The chat text will turn green when you type it fully.

<img width="508" height="71" alt="image" src="https://github.com/user-attachments/assets/50165ab3-57fb-49fb-aadc-645d55e285f2" />

Now once you hit enter, your text will be printed to chat!

:star: Error code 9009? This means that minescript cannot find your python instance. Either install it properly from the microsoft store (windows ships a link to the store instead of python by default) or manually point it to the correct instance on your PC (`where python3` -> `minecraft/minescript/config.txt`)
